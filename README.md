# Hi, I'm Anton

Audiobook recording engineer and studio coordinator in Tallinn, Estonia — and a full-stack developer who builds.

By day I record, direct, and edit audiobooks. The rest of the time I think of and build the pipelines and tools that could automate the work: an Estonian-language audiobook production platform, an NLP processing service, QA tooling, and reporting automation.

### What I'm working on
- **DigiRead Studio** — full-stack Estonian audiobook TTS platform (manuscript → ready-to-deliver audio) built on the ElevenLabs API.
- **Studio tooling** — narrator alignment & pause detection, run-tracking/reporting, and a real-time reading-position tracker.

### Tech I work with
- **Languages:** Python, JavaScript/TypeScript, Go, SQL
- **Web:** Next.js, React, FastAPI, Node.js
- **Data/Infra:** Supabase (PostgreSQL, Auth, Storage, RLS), Turborepo
- **AI/Audio:** ElevenLabs TTS, Whisper/WhisperX, NER, text normalization, ffmpeg/ffprobe
